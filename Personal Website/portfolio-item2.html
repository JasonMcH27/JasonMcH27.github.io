Features

Random World Generation

The main aspect of Dynamic Lands is the Random World generation. As you progress through the game, the environment around you is constantly evolving. There are four unique biomes that can be generated with specific presets made, these being Plains, Ice, Valley, and Void. When you start off only the easier biomes will spawn in, but as you get to higher rounds more difficult biomes will start generating.

Enemies

Each biome has a unique enemy type attached to it. As you progress through higher rounds, different variations of enemies will spawn that are stronger than their base version and drop unique buffs. In order to help the player know what that is and how strong the enemy type is we created a colored scale to help the player understand the power level of these enemy variations.

Buffs

After defeating enemies there is a chance that a random buff can drop. These buffs change the base stats of the player and also add unique traits as well. Some of these buffs include an increase to HP, adding Lifesteal, and doing damage over time.

Monolith

In order to make all main systems, this being enemy spawners, world generation, and round system work properly the Monolith was created. To start each round the player interacts with the Monolith. The Monolith then is able to generate the biomes based on the current round, and make each spawner correctly spawn in the appropriate enemy for the biome.
